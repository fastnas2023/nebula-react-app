# Nebula - Modern Video Conferencing UI

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen.svg)](https://nebula-react-app.vercel.app/)
[![React](https://img.shields.io/badge/React-18-blue.svg)](https://reactjs.org/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind-3.0-38B2AC.svg)](https://tailwindcss.com/)
[![Vite](https://img.shields.io/badge/Vite-4.0-646CFF.svg)](https://vitejs.dev/)

Nebula is a next-generation, responsive video conferencing user interface built with React, Tailwind CSS, and Framer Motion. It features a sleek glassmorphism design, real-time responsive layouts, and robust UX patterns inspired by mainstream platforms like Zoom, Microsoft Teams, and Google Meet.

## 🌟 Live Demo

Experience the live application here: **[https://nebula-react-app.vercel.app/](https://nebula-react-app.vercel.app/)**

## ✨ Key Features

- **Immersive Screen Sharing**: Double-click to fullscreen, draggable split-pane layouts, and hover-to-reveal presenter controls.
- **AI-Enhanced Chat**: Integrated mock AI assistant with smart summaries, action items, and smooth auto-scrolling thinking animations.
- **Glassmorphism Aesthetics**: Deep dark mode with dynamic mesh gradients, noise overlays, and blurred paneling.
- **Responsive Navigation**: Intelligent collapsible sidebar with tooltips (Mystery Meat Navigation avoided) and mobile-friendly bottom bars.
- **Internationalization (i18n)**: Full English and Chinese (Simplified) language support.

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/fastnas2023/nebula-react-app.git
   cd nebula-react-app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:3000` (or the port specified in your terminal).

## 🛠 Tech Stack

- **Framework**: React (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router DOM
- **i18n**: react-i18next
